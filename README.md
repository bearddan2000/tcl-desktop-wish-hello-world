# tcl-desktop-wish-hello-world

## Description
This is a POC project to demonstrate tcl gui builder
called from `wish` command.

Supports X11 display forwarding from docker container.

## Tech stack
- tcl
- wish

## Docker stack
- ubuntu

## To run
`sudo ./install.sh -u`

## To stop (optional)
`sudo ./install.sh -d`

## To see help
`sudo ./install.sh -h`
